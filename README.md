Do své složky **[styles](Lutonsky_projekt/styles/normalize.css)** jsem přidal soubor normalize.css, aby na všech prohlížečích vypadal web stejně.

V horní části stránky bude navbar s hamburger menu, který bude v  `<head>`. Pod ním pak bude začínat `<body>` který bude rozdělený <ins>následovně:</ins>
1. `<header>`
    -  galerie
2. `<main>`
    - něco o obchodu
    - `<section>`
      - karty se zbožím
3. `<footer>`
    - kontakní údaje
